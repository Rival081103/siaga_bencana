## 🚀 Cara Publish ke GitHub Pages

### Langkah 1 — Buat Repository
1. Buka [github.com](https://github.com) → Login
2. Klik tombol **"New"** (buat repository baru)
3. Nama repository: `siagaku`
4. Pilih **Public**
5. Klik **"Create repository"**

### Langkah 2 — Upload File
1. Di halaman repository, klik **"uploading an existing file"**
2. Upload semua file berikut:
   - `index.html` ← Landing page utama
   - `app.html` ← Aplikasi kuis & belajar
   - `README.md` ← Dokumentasi ini
3. Scroll ke bawah → klik **"Commit changes"**

### Langkah 3 — Aktifkan GitHub Pages
1. Klik tab **"Settings"** di repository
2. Scroll ke bagian **"Pages"** (menu kiri)
3. Di bagian **Source**, pilih: **Branch: main** → folder: **/ (root)**
4. Klik **"Save"**
5. Tunggu 1–3 menit → website aktif!

### Langkah 4 — Akses Website
```
https://USERNAME.github.io/siagaku
```
